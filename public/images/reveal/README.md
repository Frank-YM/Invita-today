# Catálogo de imágenes para revelación de género

Este es el catálogo del buscador que aparece en el admin cuando el evento es de tipo **Revelación de género**.

## Convención de nombres

`nombre-visible_tag1_tag2_tag3.png`

- La primera parte (antes del primer `_`) es el **nombre visible** en el catálogo. Usa guión medio `-` para separar palabras: `goku-nino`, `hello-kitty`, `mickey-mouse`.
- Las partes siguientes son **tags de búsqueda** separados por `_`. El buscador matchea contra cualquier tag y contra el nombre visible.

## Ejemplos

- `goku-nino_dragonball_anime_nino.png` → aparece al buscar "goku", "dragon", "dragonball", "anime", "niño"
- `hello-kitty_sanrio_kawaii_nina.png` → aparece al buscar "kitty", "sanrio", "kawaii", "niña"
- `shrek-bebe_shrek_pelicula_verde.png` → aparece al buscar "shrek", "película", "verde"

## Cómo agregar imágenes

1. Buscá o creá el PNG (ideal: fondo transparente, ~500×500px, personaje centrado).
2. Renombralo siguiendo la convención.
3. Copialo a esta carpeta.
4. Refrescá el admin — ya aparece en el buscador.

No hace falta migrar ni reiniciar nada. El sistema lee la carpeta en cada búsqueda.

## Nota legal

Si vas a usar personajes con marca registrada (Disney, Toei, Warner, etc), asegurate de tener los derechos o de usarlo solo en eventos privados personales. Este catálogo es local y no se publica automáticamente.
